# airbnb clone coding ✈


## 기능

- 로그인 

- 회원가입

- 게시글 (등록, 수정, 삭제)

- 댓글 (등록, 수정, 삭제)

## 기술스택

framework: 스프링(4.3.1)
language: java(1.6)
DB: mysql
css : bootstrap 4
server: tomcat
